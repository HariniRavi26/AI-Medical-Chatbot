# 🏥 AI Medical Appointment Chatbot

An AI-powered Medical Appointment Assistant built using Flask, SQLite, HTML, CSS, and JavaScript.

## 🚀 Live Demo

🔗 Live Application:
https://ai-medical-chatbot-1-qvza.onrender.com

## 📌 Features

- Symptom-based doctor recommendation
- Intelligent symptom matching
- Emergency symptom detection
- Doctor availability checking
- Appointment booking system
- Appointment history tracking
- Available date and time slot management
- Responsive user interface
- SQLite database integration

## 🛠️ Technologies Used

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript
- Render (Deployment)

## 💡 How It Works

1. User enters a symptom.
2. The chatbot identifies the relevant medical specialization.
3. Available doctors are displayed.
4. Users can select available dates and slots.
5. Appointments are booked and stored in the database.
